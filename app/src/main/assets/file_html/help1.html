Selamat Datang

Terima kasih sudah menggunakan aplikasi media pembelajaran Perakitan Komputer ini!. Untuk dapat menggunakan aplikasi ini secara optimal, Anda dapat menggunakan bantuan yang telah ada untuk memudahkan Anda dalam mengoperasikan aplikasi ini.

Splash Screen

Anda akan menemukan splash screen ketika Anda membuka aplikasi ini. Fungsi dari Splash screen adalah untuk menyiapkan aplikasi sehingga siap untuk digunakan.

Menu!

Setelah Anda menemukan splash screen Anda akan dibawa masuk ke dalam aplikasi. Setelah itu, Anda akan menemukan Menu!. Menu! di dalam aplikasi ini disusun sesuai dengan fungsinya yakni sebagai media pembelajaran dan dibentuk dengan gaya metro dengan tujuan agar Anda dapat lebih mudah dan cepat mengakses fitur-fitur di dalam aplikasi ini. Fitur-fitur yang ada di dalam aplikasi ini antara lain: Kompetensi, Materi, Latihan, Bantuan, dan Tentang.

Keluar dari Aplikasi

Dari tampilan Menu!, Anda dapat langsung keluar dari aplikasi. Untuk keluar, Anda hanya tinggal menyentuh tombol Back button dan otomatis aplikasi akan menampilkan konfirmasi untuk keluar. Pilih Ya jika Anda benar-benar akan keluar dari aplikas, pilih Tidak untuk membatalkannya
